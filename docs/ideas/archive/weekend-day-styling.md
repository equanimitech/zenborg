# Weekend day styling

Raw capture — 2026-05-02.

- Weekends should appear differently in the timeline and heatmap.
- Sat + Sun feel different from weekdays — UI should reflect that visually.
- Could be subtle: tinted column background, different header weight, muted gridline, or a small marker.
- Heatmap: maybe weekend cells get a softer base tone or distinct gap-space treatment.
- Timeline: weekend day headers could use a different label style (italic? lighter? small "weekend" tag?).
- Questions:
  - Stone-only (per design constraint) — how to differentiate without color?
  - Locale-aware? Some users start week on Sunday, weekends could be Fri/Sat in other cultures.
  - Does this interact with the per-day-title idea (titled days override styling)?
  - Should phase visibility differ on weekends (e.g. work-related phases auto-hidden)?
- Don't shape yet.
